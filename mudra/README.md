# crytpo

Crypto library of MOI protocol that handles 

`Key Management(KramaVault)`: Manages Keys required for signing at Consensus and P2P network communication 

`POI: Proof of Identity`: Manages KramaID, MOINode modules 

`Signature scheme`: Generic Module that manages ECDSA, Schnorr Signature schemes etc along Musig2 for Multi-Signatures
